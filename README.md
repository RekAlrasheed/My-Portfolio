# Rakan Alrasheed — Portfolio

A fast, modern, dependency-light personal portfolio.

- **Stack:** Hand-written semantic HTML, a custom CSS design system, and vanilla JavaScript — no front-end framework, no build step.
- **Features:** Dark/light theme, scroll-spy navigation, reveal-on-scroll animations, animated role typing, project spotlight cards, and a mailto-powered contact form.
- **Hosting:** Deployed on [Vercel](https://vercel.com).

## Structure

```
index.html              # Single-page site
assets/css/styles.css   # Design system + components
assets/js/main.js       # Interactions
assets/pdf/             # CV
assets/favicons/        # Icons & manifest
vercel.json             # Static hosting config
```

## Local preview

```bash
python3 -m http.server 8080
# open http://localhost:8080
```
